    Gitlet

    Gitlet is a recreation of the known-and-loved version control system, Git. This project, programmed in Java, implements all of the features of Git: local and remote repositories, branches, commits, merges, commit checkouts, branch checkouts, and more. Persistence, serialization, hashing, and file manipulation are just a few of the concepts that I enjoyed through this project. The TreeMap data structure was utilized to not only track commits, but also do so in optimal time.

    Parking Meters

    **IN PROGRESS**

    This project utilizes real-time data via the LADOT public database to show a map of all available parking meter spots at any given time. Data at time of request is stored in a PostgreSQL database to allow for quicker access and improved data analysis. Eventually, this program will show all available spots in an area specified by the user.

    Ataxx AI

    Ataxx is two player strategy board game played on a 7x7 board. Players alternate turns either jumping or extending pieces to capture the opponent's pieces. Utilizing the object-oriented nature of Java, this project incorperates stacks and arraylists to develop command-line and GUI versions of Ataxx with optimized time complexity. For single-player mode, a strong AI is implemented using game trees and alpha-beta pruning algorithms.
